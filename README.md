KeyButtons:
-
>[!IMPORTANT]
> `<Space+q>` quit \
> `<Ctrl+a>` copy all \
> `<Ctrl+s` save \
> `<Ctrl+n>` toggle nvim-tree \
> `<Ctrl+f>` focus on nvim-tree \
> `<Tab>` cycle selection next buffer tab \
> `<Shift+Tab>` cycle selection prev buffer tab \
> `<Ctrl-b>` close other opened buffer tabs \
> `<Ctrl+t>` toggle highlighting illuminate \
> `<Ctrl+p>` telescope find files \
> `<Space+Space>` telescope old files\
> `<Space+f+g>` telescope live grep \
> `<Space+f+h>` telescope help tags \
> `<Space+g+b>` telescope git branches \
> `<Space+g+c>` telescope git commits \
> `<Space+g+s>` telescope git tatus \
> `<Space+c+s>` telescope colorscheme \
> `<M>` line comment \
> `<N>` block comment \
> `<K>` Show info \
> `<Space+r+n>` lsp rename \
> `<Space+c+a>` lsp code action \
> `<Ctrl+b>` scroll docs back \
> `<Ctrl+f>` scroll docs forward \
> `<Ctrl+e>` close docs \
> `<Space+Enter>` confirm \
> `<Tab>` autocomplete select next \
> `<Shift+Tab>` autocomplete select prev \
> `<j+k>` fast escape to normal mode
